Welcome to the Year 1999

When HTML and CSS ruled the interwebs.

It is the year 1999. The internet is a thing. But memes are not a thing. Social media is not a thing. Streaming video is not a thing. Infinite scrolling is not a thing. Browser apps are definitely not a thing.

To access the internet, you have to use dial-up. This used your landline connection.

The fastest internet speed ran at about 56Kbs, roughly 0.01% the speed you're used to.

If a single page had a fair amount of content, it could take up to five minutes to load.

The Internet in the 1990s

  • Browsers: Netscape, Internet Explorer
  • Email and messengers: AOL, Geocities, AIM, MSN Messenger
  • Search Engines: Yahoo!, AltaVista, Lycos

How to get onto the internet in 1990

  1. Buy a Dial-up Modem and a Phone Line
  2. Install Dial-up Modem into Computer
  3. Connect Modem to phone line in house
  4. Make sure no one is using the phone line
  5. Start-up modem to connect via phone line
  6. Wait. . . and listen to this sound
  7. Yay! Connected. Time to start surfing the web!
