# HTML, CSS & JS Lessons

A collection of lessons covering the basics of HTML, CSS, and JS.

## Setup in VSCode / Codespaces

### Required Extensions

* **Live Server** – Necessary for previewing your pages.

  * Start by clicking the **“Go Live”** button at the bottom-right of VSCode.

### Recommended Extensions

* **Prettier** – Formats your code automatically.
* **Stylelint** – Helps catch CSS errors and enforce coding style.
* **Auto Close Tag** – Automatically closes HTML tags for faster coding.

## Online Playgrounds

Use these to experiment with HTML/CSS/JS without installing anything:

* [MDN Web Playground](https://developer.mozilla.org/en-US/play)
